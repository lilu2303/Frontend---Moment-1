Om uppgiften

Den här uppgiften går ut på att jag ska med hjälp av NPM, parcel och Nodejs ska skapa en automatiserad hemsida som uppdateras och laddas upp till Netlify. För att göra så att alla ändringar automatiskt uppateras vid nedsparning i vsc, skapade jag först ett github repo där jag sparade ner alla ändringar av koden i vsc. Och när hemsidan sedan var färdig länkade jag mitt github repo med Netlify där alla ändringar sedan publiceras automatiskt vid uppatering av gitrepot.



Arbetsprocess

Vid utveckling av hemsidan skriver jag den kod som jag tycker är lämplig för hemsidan och för dess design, och när jag sedan känner färdig öppnar jag komandotolken och skriver "NPM run start" för att starta parcel projektet och testköra arbetet i en så kallad liveserver. Därifrån kan jag sedan göra förbättringar och sedan när jag känner mig nöjd kan jag commita och pusha commitsen till github som sedan är kopplat till Netlify, för att på så sätt automaticera publiceringen av min hemsida.
